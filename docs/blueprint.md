# **App Name**: Dijital Andac

## Core Features:

- Admin Panel: Admin panel for managing schools.
- Add School: Admin can add unlimited schools via the admin panel.
- Student Login: Student login on the modern designed homepage.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) -> RGB(#3399FF) for a modern and trustworthy feel.
- Background color: HSL(210, 20%, 95%) -> RGB(#F0F8FF) for a clean and calming interface.
- Accent color: HSL(180, 60%, 40%) -> RGB(#33CCCC) for interactive elements and highlights.
- Headline font: 'Poppins', sans-serif, for a contemporary and geometric look.
- Body font: 'PT Sans', sans-serif, for readability and a modern touch.
- Use clean, minimalist icons to represent different functionalities within the platform.
- A clean, modern layout with ample spacing and clear hierarchy.
- Subtle transitions and animations to enhance user experience.
